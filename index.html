Network Flows Analyzer

This is a Directed Research project done by Dipankar Patro under Prof. Mike Zyda.

This web-application presents how d3.js is used to visualize the characteristics of flows in a Computer Communication Network.

How to use:

  1. Select a data set to load from the selector given below.
  2. A graph will show up having IP addresses as nodes and Flows as edges between them. The more the color intensity on an edge, the more communication between those IPs.
  3. Click on any node to show the ingress flows on the IP in a table below the graph.
